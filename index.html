Using vue.js to build the first simple website

参照官方文档的NPM方法（MAC）：
1、安装/升级NPM：
（1）查看NPM版本：npm -v
(2) NPM版本更新：npm install npm -g
2、安装Vue：
npm install vue
3、使用命令行工具快速搭建Vue单页应用：
# 全局安装 vue-cli
$ cnpm install –global vue-cli
# 创建一个基于 webpack 模板的新项目
$ vue init webpack my-project
# 这里需要进行一些配置，默认回车即可
This will install Vue 2.x version of the template.

For Vue 1.x use: vue init webpack#1.0 my-project
